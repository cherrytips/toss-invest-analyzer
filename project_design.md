현재 프로젝트 폴더 내에서 다음과 같은 서비스를 만들려고 해.
너는 주식시장 10년차의 베테랑 증권 분석가야.

해당 서비스는 로컬에서 동작하고, 구동 시 분석 결과를 출력해주는 서비스야.
프로젝트 구조를 설계하고 구현해줘.

[ 토스증권의 관심종목 분석 도구 ]

# 각 사용자의 토스증권 관심종목 기준으로 아래 서비스를 수행함.

1. 토스증권에 로그인이 필요함
웹 주소 : https://www.tossinvest.com/
2. 사용자 정보는 UserInfo.md 파일로 별도 저장하여, 해당 서비스를 이용하는 사용자 정보를 입력함
정보 : 이름/생년월일/전화번호

# 아래 정보는 토스 증권에 로그인하여 웹에서 받아와야 함

1. 일일 시장 분석 - 사용자 관심 종목의 당일(현재 시점) 주요 뉴스, 시황, 투자 포인트를 분석
2. 매수/매도 타이밍 예측 - 지표 + 실시간 뉴스를 종합한 예측 실행
3. 예측은 아래 기준을 따름.
단기 (1주 ~ 4주): 모멘텀, 거래량, 단기 뉴스 기반
중기 (1달 ~ 6달): 실적, 섹터 트렌드, 매크로 기반
장기 (1년 ~ 3년): 펀더멘털, 산업 구조, 성장성 기반
4. 통계 대시보드 — 등락률, 거래량 추이, 섹터별 분포 시각화
5. 현재 시점의 각 종목별 호재, 악재 뉴스 요약 / 기술적 분석
6. 현재 시점 종합 투자 의견
